# Task: Chat fragment - session list

## Acceptance Criteria

### Session List Item
```rust
use ratatui::widgets::ListItem;

fn session_to_list_item(session: &Session) -> ListItem<'static> {
    ListItem::new(format!(

        session.title,
        session.updated_at.format("%H:%M")
    ))
}
```
