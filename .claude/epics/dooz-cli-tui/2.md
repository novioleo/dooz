# Task: Project scaffolding

## Description

Create the dooz_cli Rust project with cargo, set up the directory structure, configure dependencies, and implement basic TEA (The Elm Architecture) skeleton with terminal setup.

## Acceptance Criteria

- [ ] `cargo new dooz_cli` creates project at project root
- [ ] Cargo.toml has all required dependencies
- [ ] Basic project structure matches epic design
- [ ] Terminal setup with ratatui initializes correctly
- [ ] Error handling with color-eyre installed
- [ ] Basic TEA loop skeleton compiles

## Technical Details

### Create Project
```bash
cargo new dooz_cli
```

### main.rs Pattern
```rust
use color_eyre::Result;
use ratatui::Terminal;

fn main() -> Result<()> {
    color_eyre::install()?;
    let mut terminal = ratatui::init()?;
    let app = App::new();
    run_app(&mut terminal, app)?;
    ratatui::restore_terminal();
    Ok(())
}
```
