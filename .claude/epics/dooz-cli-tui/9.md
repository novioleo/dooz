# Task: Chat fragment - commands

## Description

Implement `/new` and `/exit` commands: `/new` creates a new session and switches to it, `/exit` terminates the entire program.

## Acceptance Criteria

- [ ] `/new` creates new session with timestamp title
- [ ] `/new` switches to new session
- [ ] `/exit` exits program cleanly
- [ ] Commands detected in input when starting with `/`
- [ ] Unknown commands show error or ignore
- [ ] Session auto-created on program start

## Technical Details

### Command Detection
```rust
fn handle_input(input: &str) -> Option<Action> {
    match input.trim() {
        "/new" => Some(Action::CreateSession),
        "/exit" => Some(Action::Exit),
        _ => None,
    }
}
```

### Action Enum Updates
```rust
#[derive(Debug, Clone)]
pub enum Action {
    // ... existing actions
    CreateSession,
    Exit,
}
```

### Create Session Flow
```rust
fn create_session(model: &mut ChatModel, store: &SessionStore) {
    let title = Utc::now().format("%Y-%m-%d %H:%M").to_string();
    let session = store.create_session(title).unwrap();
    model.active_session = Some(session);
}
```

### Exit Flow
```rust
fn exit(app: &mut App) {
    app.should_quit = true;
}
```

### Startup Session
```rust
impl App {
    fn new() -> Self {
        let store = SessionStore::new().unwrap();
        let session = store.create_session(
            Utc::now().format("%Y-%m-%d %H:%M").to_string()
        ).unwrap();
        let mut app = App { store, ..Default::default() };
        app.active_session = Some(session);
        app
    }
}
```
