# Task: Chat fragment layout

## Description

Build the 3-panel layout for Fragment1 (Chat): left session list (20% width), right top chat history (80% width, 80% height), right bottom input area (80% width, 20% height). Implement the layout structure with borders separating each panel.

## Acceptance Criteria

- [ ] Left panel: Session list (20% width, full height minus borders)
- [ ] Right top: Chat history (80% width, 80% height)
- [ ] Right bottom: Input area (80% width, 20% height)
- [ ] All three panels have visible borders
- [ ] Layout adapts to terminal resize
- [ ] Minimum terminal size handled (80x24)

## Technical Details

### Layout Calculation (src/ui/layout.rs)
```rust
use ratatui::{Layout, Constraint, Rect};

pub fn chat_layout(area: Rect) -> [Rect; 3] {
    Layout::vertical([
        Constraint::Fill(1),
    ])
    .areas(area);
    
    // Split horizontally: 20% left, 80% right
    let [left, right] = Layout::horizontal([
        Constraint::Percentage(20),
        Constraint::Percentage(80),
    ]).areas(area);
    
    // Split right vertically: 80% top, 20% bottom
    let [chat_area, input_area] = Layout::vertical([
        Constraint::Percentage(80),
        Constraint::Percentage(20),
    ]).areas(right);
    
    [left, chat_area, input_area]
}
```
